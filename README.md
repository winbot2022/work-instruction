# 作業手順書作成アプリ

## 概要

製造現場向けの、写真付き作業手順書を簡単に作成するためのStreamlitアプリです。

工程を選ぶと標準的な作業手順テンプレートが表示され、必要な手順を選択・追加し、写真・ポイント・注意事項・確認項目を登録できます。

出力はExcel形式です。

---

## コンセプト

作業手順書をゼロから書かせないことを重視しています。

- 工程を選ぶ
- 標準手順を選ぶ
- 必要な写真を入れる
- 注意点や確認項目を選ぶ
- Excel手順書を出力する

現場での文字入力をできるだけ少なくし、写真と選択入力を中心に作成できます。

---

## 主な機能

- 工程別の標準手順テンプレート表示
- 手順の選択・削除
- 手順の追加
- 手順ごとの写真登録
- ポイント候補の選択
- 注意事項候補の選択
- 確認項目候補の選択
- 自由入力による追記
- Excel出力

---

## 初期工程テンプレート

以下の工程テンプレートを用意しています。

- 汎用テンプレート
- マシニング加工
- NC旋盤加工
- プレス加工
- バリ取り
- 外観検査
- 寸法測定
- 組立
- 洗浄
- 梱包
- 日常点検

---

## Excel出力内容

出力Excelには、以下のシートを作成します。

### 1. 作業手順書

作業手順書本体です。

- 基本情報
- 小写真付きの手順一覧
- 作業手順
- ポイント
- 注意事項
- 確認項目

### 2. 写真集

登録した写真を大きく表示する確認用シートです。

### 3. 入力データ

後から編集・確認しやすいように、入力内容を一覧化したシートです。
